# Paper: Parallel array operations in Java
This GitHub contains a scientific paper and the corresponding demos. Furthermore, the title of the paper is: _**Swiftly parallelizing sequential array manipulations in Java with standard library methods for increased performance**_. 
This paper is an assignment of the course IT-Applications at the University of Applied Sciences Offenburg (Germany). Due to this circumstance, the presentation will only be available in German.
