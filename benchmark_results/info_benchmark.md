# Info for files in this folder
This folder contains the JSON results of the benchmark on my maschine, which can be viewed with the [JMV visualizer](https://jmh.morethan.io/) as bar graphs. For now only the newest results will be saved.

The output as JSON file is configured in the runtime configuration of in IntelliJ with the instruction:   
`-rf
json
-rff
benchmark_results/ArraySortBenchmarkResults.json`
